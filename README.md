# DNC程式傳輸管理追蹤台

集中管理與傳輸CNC加工程式至現場機台之系統，確保加工程式版本正確一致。

- 原始系統：DNC程式傳輸管理系統（Distributed Numerical Control）
- 產業分類：智慧製造｜製造管理
- 使用部門：製造工程、生產、資訊部門
- 工作流程：程式上傳版控 → 機台配對確認 → 程式傳輸執行 → 加工執行 → 傳輸紀錄歸檔
- AI 應用：AI偵測異常程式版本使用、智慧建議程式優化參數
- 主要 KPI：程式錯誤傳輸事件數、傳輸成功率、版本一致率
- 代表產品：Predator Software（Predator DNC）、CIMCO DNC-Max
